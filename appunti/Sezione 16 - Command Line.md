# Command Line
```md
dir = mostra tutti i file nella directory principale
cd nome_cartella = entra nella cartella
cd .. = va nella cartella precedente
clear = pulisce il terminale
(key tab) = auto completamento
touch nome_file = crea un nuovo file
rm nome_file = elimina il file
mkdir nome = crea una nuova cartella
rmdir nome = elimina la cartella
```
